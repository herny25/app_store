# RetailFlow — POS & Inventory Management

App móvil completa de punto de venta e inventario.
Expo SDK 53/54 + React Native + SQLite local + Zustand.

## Instalación

```bash
npm install
npx expo start
```

## Stack
- expo ~53.0.0
- expo-router ~4.0.0
- expo-sqlite ~15.0.0
- react-native 0.76.7
- zustand ^4.5.5
- @expo-google-fonts/dm-sans + syne
